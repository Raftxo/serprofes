# Proyecto: Galería de Pintores (Reto - Challenge semana 3 curso serProfés)

## ✔️ Objetivo
Crear una galería con páginas individuales para Goya, Greco, Rubens y Velázquez.

## 📁 Estructura
- `index.html` — página principal
- `goya.html`, `dynamicGreco.html`, `rubens.html`, `velazquez.html`
- `assets/images` — imágenes organizadas por pintor (también contiene el archivo JSON)
- `assets/styles` — estilos CSS

## 🧠 Mejoras pendientes
- [ ] Completar el archivo JSON con la lista de todas las imágenes
- [ ] Cargar las imágenes dinámicamente con JavaScript (de momento solamente implementado en dynamicGreco.html)
- [ ] Unificar las páginas en una plantilla única
